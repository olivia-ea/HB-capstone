SongMuse

Your activity based playlist generator.

SongMuse uses the Spotify API to generate playlists based off the user's desired activity. Once created, the user can view and listen to the playlist through SongMuse or Spotify itself, as it seamlessly uploads there. The user's previously created playlists are also saved and available for replay. Lastly, new users require a one-time Spotify authorization prior to using SongMuse.
